# <Prework Study Guide Webpage >

## Description

A study guide for use for prework for the coding bookcamp that started in 12/11/2023 and for 3 months for full stack web development. Created to assist in studying topic areas necessary for success in the course, HTML, CSS, GIThub and Javascript.


## Installation

N/A

## Usage

The prework study guide lists four topics of study, HTML, CSS, Github and Javascript and notes taken for the prework portion of the coding bootcamp on those topics. Open the DevTools Console and you will see all four topics listed and a random one chosen to review and study. Refresh to open another topic chosen at random to study.
